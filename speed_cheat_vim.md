the basic:
----------


word
----
* dw - delete from cursor to next
* db - delete from cursor to previous
* diw - delete inside word
* `*` -
* `#` -
* `%` -

for html/jsx/xml/...
--------------------
* remember `*` `#` and `%` 
* `cat` - change a tag
* `cit` - change inner tag
* `dat` - delete a tag
* `dit` - delete inner tag
* `vat` - visual a tag
* `vit` - visual inner tag
* `vitp` - visual inner tag past
* `yat` - yank a tag
* `yit` - yank i tag
